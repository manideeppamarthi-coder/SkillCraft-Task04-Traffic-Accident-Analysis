# SkillCraft Data Science Internship – Task 04

## Traffic Accident Data Analysis

### Objective

Analyze traffic accident data to identify patterns and trends related to accident occurrence, severity, and contributing factors.

### Dataset

Traffic Accident Dataset (`traffic_accident_dataset_500.xlsx`)

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Jupyter Notebook

### Project Workflow

1. Loaded and inspected accident records.
2. Performed data cleaning and preprocessing.
3. Analyzed accident frequency and severity.
4. Examined relationships between accident factors.
5. Created visualizations to highlight important trends.

### Key Skills Demonstrated

* Data Cleaning
* Exploratory Data Analysis
* Statistical Analysis
* Data Visualization
